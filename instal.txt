УСТАНОВКА

1.Установка пакета модифицирующего Delphi (находится в папке ./D7_style/)
	1.1.Папка с компонентами должна быть зарегистрирована в Delphi. 
		(Tools->Environment Options->Library->Library Path)
	1.2.Открываем в Delphi файл acntd7_r.dpk.
	1.3.В открывшемся окне нажимаем "Build".
	1.4.Открываем файл acnt7.dpk.
	1.5.В открывшемся окне  нажимаем "Install".
	1.6.Открываем файл AlphaDB7.dpk.
	1.7.В открывшемся окне  нажимаем "Install".
	
2.Создание базы данных из скрипта CREATE_BD.sql.
	(Проверена раюота на MS SQL Server 2008 R2 и MS SQL Server 2014)
	2.1.Открываем SQL Server.
	2.2.Выбираем в меню Файл->Открыть->Файл, далее выбираем местоположение
		файла (по умолчанию ./SQL/CREATE_BD.sql) и открываем его.
	2.3.Далее нажимаем "Выполнить", база данных создана.

	
ОПЦИИ (Хранятся в ini-файле, по умолчанию ./program/Config.ini)
1.[DataBase]-> IPDef - хранит адресс MSSQLServer`а который будет использоваться по умолчанию. 
	(также происходит поиск доступных устройств в локальной сети)
2.[Auth]-> UserLogin - хранит логин авторизовавшегося вошедшего пользователя, c этого устройства. 
	(используется при входе чтобы упростить повторную авторизацию пользователя)
3.[Design]-> SkinID - хранит номер стиля который был выбран последним (для авторизации).
4.[StyleID]-> <Имя пользователя> - хранит номер стиля который выбрал данный пользователь.
